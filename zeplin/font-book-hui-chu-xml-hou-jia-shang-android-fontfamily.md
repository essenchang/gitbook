# Font Book 匯出後加上 android:fontFamily

Zeplin 匯出 Style 並不會加上 fontFamily 屬性，必須自行手動加入

&lt;item name="android:fontFamily"&gt;...&lt;/item&gt;

```
sans-serif            # Regular
sans-serif-light
sans-serif-condensed
sans-serif-black
sans-serif-thin
sans-serif-medium
```
